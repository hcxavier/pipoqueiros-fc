# pipoqueiros-fc

## Como rodar a API

1. Certifique-se de estar na branch `mobile` e atualize o código:

    ```bash
    git checkout mobile
    git pull
    ```

2. Entre no diretório da API, instale as dependências e inicie o projeto:
    ```bash
    cd api
    npm install
    npm run dev
    ```

## Documentação da API

Para acessar a documentação da API Swagger/Scalar, abra o navegador no endereço:
[http://localhost:3333/api-docs](http://localhost:3333/api-docs)
